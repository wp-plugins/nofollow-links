Select which links in your blogroll you want to nofollow.

== Description ==

This is a plugin designed to allow you to append nofollow to the rel attribute
of selected links in your blogroll.

It does not conflict with the link relationships (XFN) specified when you edit
a link.  It will just be appended to the end of the rel attribute when it's
displayed.

== Installation ==

1. Upload `nofollow-links` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Manage > Nofollow Links and select links you want nofollowed
